# Projeto X-Men

O Projeto X-Men é uma aplicação interativa desenvolvida durante as aulas do curso Devemdobro. A página de seleção de agentes X-Men oferece uma experiência dinâmica e informativa. Ao passar o mouse sobre o rosto de um personagem, uma área à direita da tela exibe automaticamente uma versão ampliada da imagem, acompanhada do nome do personagem e uma breve descrição sobre ele. Essa funcionalidade permite aos usuários conhecer mais sobre cada agente de forma intuitiva e envolvente.
